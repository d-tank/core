Welcome to dtank. Dtank is a rapidly deployable, distributed, fault-tolerant file storage system.

Designer Templates

Dtank is based primarily on MooseFS, but has been adapted to run inside of Docker virtual machine containers [with persistent storage on a Docker host's local storage, leveraging etcd for distributed management of many nodes.

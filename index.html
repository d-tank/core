welcome to dtank

Welcome to dtank. Dtank is a rapidly deployable, distributed, fault-tolerant file storage system.

Qu'est-ce que Dtank?

Dtank is based primarily on MooseFS, but has been adapted to run inside of Docker virtual machine containers [with persistent storage on a Docker host's local storage, leveraging etcd for distributed management of many nodes.

uh...huh...

D-Tank is also the name of a viral bioweapon that ultimately forms the crux of the 2004 feature film Appleseed, as well as the name of the freaking massive walking tank in that movie. It's pretty bad-ass. Dtank was chosen as a name as the system is distrubuted, uses container technology, and aims to hold all yo' data. Dtank totally doesn't want to kill you.

How can I use it?

Dtank is currently being written. As releases become available for testing, they'll be linked to on here both as source and as pullable docker machines.
